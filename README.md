# Eye by Proctorme — Admin Web App

**Eye by Proctorme Admin** is the central dashboard for managing and monitoring all proctoring activities within the Proctorme ecosystem.  
It allows organizations, exam administrators, and partners to easily manage accounts, configure settings, and access proctoring insights in one place.

## Key Features

- **User Authentication** – Secure login and account management for verified users.
- **Subscription & Payments** – Manage billing, plans, and renewals directly from the dashboard.
- **Widget Integration** – Copy and customize your **Proctorme Widget Embed Code** to seamlessly integrate proctoring into your own platform.
- **Exam Monitoring** – View candidate activity, flags, and event summaries in real-time.
- **Secure API Access** – Request API keys and manage connected domains for verified integrations.

## Getting Started

1. Sign up or log in with your organization credentials.
2. Configure your webhook URLs, API keys, and widget settings.
3. Copy your unique widget embed code and start monitoring exams instantly.

## How to start developing

- create .env file and copy all from .env.example
- pnpm install
- npm run dev
